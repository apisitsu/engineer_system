ภาพจำลองเรขาคณิต (CAD View)

พารามิเตอร์กำหนดเงื่อนไข (ปรับค่าได้)

ผลการคำนวณรัศมี (R)

รัศมีเป้าหมายที่คำนวณได้:

R 21.8477

ขั้นตอนที่ 1: หาสมการเส้นสัมผัส

เส้นสัมผัสทำมุม 35° ลาดลง เวกเตอร์ตั้งฉาก (Normal Vector) จะทำมุม α = 90° - 35° = 55°

สมการเส้นตรงที่สัมผัสวงกลม D (รัศมี R₁ = 13.02) คือ:

x·cos(α) + y·sin(α) = R₁
x(0.5736) + y(0.8192) = 13.02

ขั้นตอนที่ 2: หาจุดศูนย์กลางวงกลมเล็ก (Xc, Yc)

วงกลมเล็ก (R₂ = 3.2) สัมผัสเส้นระดับ H = 4.455
ดังนั้น Yc = H + R₂ = 7.655

ระยะห่างจาก (Xc, Yc) ถึงเส้นสัมผัสในข้อ 1 ต้องเท่ากับ R₂:

Xc·cos(α) + Yc·sin(α) - R₁ = R₂
Xc = [R₁ + R₂ - Yc·sin(α)] / cos(α)
Xc = [13.02 + 3.2 - (7.655 × 0.8192)] / 0.5736
Xc = 17.346

ขั้นตอนที่ 3: คำนวณรัศมีเป้าหมาย (R)

วงกลม R สัมผัสวงกลม D ที่ยอดสุด พิกัดศูนย์กลางคือ C(0, R₁ - R)
และเส้นรอบวงต้องลากผ่านจุดสัมผัสระหว่างวงกลมเล็กกับเส้น H ที่พิกัด P(Xc, H)

ตั้งสมการระยะทางจาก C ถึง P:

R² = (Xc - 0)² + [H - (R₁ - R)]²
ให้ระยะ Δy = R₁ - H = 8.565
R² = Xc² + (R - Δy)²
R² = Xc² + R² - 2R(Δy) + Δy²
2R(Δy) = Xc² + Δy²
R = (Xc² + Δy²) / (2 × Δy)
R = (17.346² + 8.565²) / (2 × 8.565)
R = 21.8477
